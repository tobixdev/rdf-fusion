This folder contains data files from [1] that otherwise must be downloaded everytime from Zenodo during the CI build.
Sometimes this download fails, and therefore we include the files here. The files are licensed under the Apache License 
2.0. The copyright is with the original authors of [1].

The files currently only include the recorded queries files. The datasets themselves are generated during the CI build.

[1] T. Pellissier Tanon, „BSBM v3.1 SPARQL benchmark pre-generated files“. Zenodo, Juli 06, 2024.
    doi: 10.5281/zenodo.12663333.
